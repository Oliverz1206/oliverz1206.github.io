## 0. 前言

这篇笔记用于整理 **Mac 内建触控板** 与 **Magic Trackpad** 的常见系统级手势，方便后续在个人网页中长期留档、检索和查看。

需要注意的是：

- 不同 macOS 版本中，部分手势可能显示为 **三指** 或 **四指**。
- 某些功能需要在 **系统设置 -> 触控板** 中手动开启。
- **Force Click（用力点按）** 仅适用于支持 Force Touch 的机型。
- **三指拖移** 默认通常不开启，需要到辅助功能中单独打开。

> 这份清单主要整理的是 **系统级、最常用、最值得记忆** 的触控板操作，而不是某个单独 App 内的私有手势。
{: .prompt-info }

## 1. 基础点击与选择

| 分类     | 手势            | 作用             | 备注                                     |
| -------- | --------------- | ---------------- | ---------------------------------------- |
| 点击     | 单指轻点        | 点按             | 需要开启“轻点来点按”                     |
| 点击     | 单指按下        | 普通点击         | 即传统点击                               |
| 右键     | 双指点按 / 点击 | 辅助点按（右键） | 最常用的右键方式                         |
| 查找     | 三指轻点        | 查询与数据检测   | 可对单词、地址、日期、电话等触发快速操作 |
| 强力点按 | 单指用力按压    | Force Click      | 可用于查词、预览、快速查看等             |

## 2. 滚动、翻页与浏览

| 分类     | 手势             | 作用            | 备注                              |
| -------- | ---------------- | --------------- | --------------------------------- |
| 滚动     | 双指上下滑动     | 上下滚动页面    | 方向受“自然滚动”设置影响          |
| 横向滚动 | 双指左右滑动     | 左右滚动内容    | 常见于表格、时间轴、宽页面        |
| 翻页     | 双指左滑 / 右滑  | 前一页 / 后一页 | 常见于 Safari、预览、部分文档界面 |
| 通知中心 | 从右边缘向左滑入 | 打开通知中心    | 必须从边缘开始滑动                |

## 3. 缩放与旋转

| 分类     | 手势         | 作用            | 备注                      |
| -------- | ------------ | --------------- | ------------------------- |
| 缩放     | 双指张开     | 放大            | 图片、网页、PDF 中最常用  |
| 缩放     | 双指捏合     | 缩小            | 与放大相对                |
| 智能缩放 | 双指双击     | 智能放大 / 缩回 | 多用于网页和 PDF          |
| 旋转     | 双指相对旋转 | 旋转对象        | 仅在支持旋转的 App 中可用 |

## 4. 桌面、窗口与空间管理

| 分类                 | 手势               | 作用                       | 备注                         |
| -------------------- | ------------------ | -------------------------- | ---------------------------- |
| Mission Control      | 三指或四指向上滑   | 打开调度中心               | 取决于系统版本和设置         |
| App Exposé           | 三指或四指向下滑   | 查看当前 App 的所有窗口    | 适合一个应用开很多窗口时使用 |
| 切换桌面 / 全屏空间  | 三指或四指左右滑动 | 在多个桌面与全屏应用间切换 | 非常高频                     |
| 显示桌面             | 拇指 + 三指张开    | 临时显示桌面               | 常被理解为“五指放开”         |
| Launchpad / 应用总览 | 拇指 + 三指捏合    | 打开应用总览               | 很多人会误以为这是 Spotlight |

> 需要区分：**应用总览（Launchpad）** 和 **Spotlight 搜索** 不是一回事。触控板捏合通常对应的是应用总览，不是聚焦搜索。
{: .prompt-warning }

## 5. 拖拽与窗口移动

| 分类     | 手势           | 作用                       | 备注                           |
| -------- | -------------- | -------------------------- | ------------------------------ |
| 拖移     | 三指拖移       | 拖动窗口、文件、文本、选区 | 需要在辅助功能中单独开启       |
| 选中文本 | 按住后拖动     | 选择文本范围               | 可配合轻点、按下或三指拖移理解 |
| 拖拽文件 | 按住文件后移动 | 移动文件 / 图标            | Finder、桌面中常用             |

## 6. 适合长期记忆的高频手势

如果只保留最值得记住的一组，那么建议优先记下面这些：

| 优先级 | 手势               | 用途                |
| ------ | ------------------ | ------------------- |
| 高     | 双指上下滑动       | 滚动页面            |
| 高     | 双指点按           | 右键菜单            |
| 高     | 双指张开 / 捏合    | 放大与缩小          |
| 高     | 三指或四指左右滑动 | 切换桌面 / 全屏应用 |
| 高     | 三指或四指向上滑   | Mission Control     |
| 高     | 三指或四指向下滑   | App Exposé          |
| 高     | 拇指 + 三指张开    | 显示桌面            |
| 中     | 拇指 + 三指捏合    | 打开 Launchpad      |
| 中     | 从右边缘向左滑入   | 打开通知中心        |
| 中     | 三指拖移           | 拖动窗口和对象      |
